# 制备类实验

## 一.原料预处理

### 1. 研磨/粉碎

处理：

- 底部高速气流.
- 球/研磨、超声
- 乳化—十二烷基磺酸钠（SDS）等

目的：

1. 增大物质之间接触面积，加快反应(或溶解)速率；【单位时间内增加反应速率】
2. 增大产率(或浸取率)【推动平衡抵达；提高单位时间内的产率】

### 2. 搅拌

使反应物充分接触，加快反应（或溶解）速率

### 3. 煅烧(灼烧、焙烧)

处理：

1. 空气中灼烧：氧化核心元素(或杂质)。
   $\ce{Fe^{2+} \rightarrow Fe^{3+}, S^{2-} \rightarrow SO2 \uparrow}$
2. 加入焦炭，高温灼烧：还原核心元素(或杂质)
3. 焙烧水浸法 $\ce{FeCl3,MnCl2(s) ->T[焙烧] MnCl2,Fe2O3 ->T[水浸]Fe2O3(s)+MnCl2(aq)}$

目的：

1. 不易转化物质 $\rightarrow$ 容易转化物质 比如 $\ce{Cu -> CuO}$
2. 矿物 $\rightarrow$ 氧化物
   1. 单质 $\rightarrow$ 氧化物
   2. 盐 $\rightarrow$ 分解 $\rightarrow$ 氧化物
3. 除去可燃性、热稳定性差【化学性质】、易挥发【物理性质】或有机杂质【分子晶体+能烧】

常见的易分解【化学性质】的物质：  
铵盐、结晶水合物、$\ce{H2O2、 NaHCO3、 Ca(HCO3)2、KMnO4}$

常见的挥发【物理性质】的物质：  
$\ce{Br2(l)、I2(s)}$（易升华）、$\ce{HCl、HNO3、CH3COOH、C2H5OH、}$有机物

### 4. 浸出

目的：使原料中的特定物质溶解(物理溶解或化学反应)，可溶性离子进入溶液，不溶物通过过滤除去

#### 1. 水浸

与水反应【$\ce{CuO}$】或溶解【$\ce{NH4Cl}$】

#### 2. 酸浸

1. 溶解(含 $\ce{Si}$ 的物质为滤渣)，将原料中的某些物质转化为离子
   。$\ce{SiO2 ->T[H^+][不反应], R(SiO3)n ->T[溶]SiO3^{2-}->T[H^+]H2SiO3}$
2. 除去氧化物(膜)。$\ce{Al2O3,Fe2O3,CuO}$
3. 提供酸性环境（提供 $\ce{H^+}$ 、调节 pH、促进/抑制水解）【含有弱碱金属阳离子
   （$\ce{Al^3+,Fe^2+,…}$）的溶液酸性条件加抑制水解】

#### 3.碱浸

1. 溶解 $\ce{Al、SiO2}$
   
   - 工业生产中 $\ce{Al,Al2O3 ->T[OH^-]AlO2^-  + CO2 + 2H2O= Al(OH)3}$

2. 除去氧化物薄膜($\ce{Al2O3}$)。

3. 除去油污【油脂在碱性条件下的完全水解】

4. 提供碱性环境（阴阳离子水解）

#### 4. 有机物浸

1. 萃取溶解或者除去与醇等有机物反应的物质
2. 除去油污【相似相溶】

#### 增大浸出率的措施

1. 减小 ×× 的粒子直径，增大接触面积，使物质反应充分
2. 适当改变反应温度（涉及平衡时考虑 $\Delta H$）
3. 适当增大浸出剂溶液的浓度【推动平衡正向移动；增大反应速率】
4. 加入过量的浸出剂，并不断搅拌
5. 延适当长浸出时间使反应充分进行
6. 及时分离生成物
7. 加乳化剂，增大难溶物的溶解度

## 二.物质转化

### 1. 加入某种物质的作用

#### 1.沉淀剂

1. 复分解。例：$\ce{Na2CO3、Na2S、NaOH}$ 等。除去 $\ce{Ca^{2+}, Ba^{2+}, Mg^{2+}, Cu^{2+}}$
   
   - 判断是双水解还是复分解：$\ce{RnCO3}$ ，若 $\ce{Rn^2+}$ 和 $OH^-$ 更强，那 $\ce{OH^-}$ 是 可以由
     $\ce{CO3^{2-}}$ 水解来的，所以是双水解

2. 双水解。例：$\ce{Na2CO3、Na2S}$ 等。除去 $\ce{Fe^{3+}, Al^{3+}}$【先记水解产物再物质配平+水】

3. 金属离子除杂
   
   1. 高价 (+3,+4) 常通过水解的方法转化为难溶氢氧化物除杂。
   
   2. 二价金属常用 $\ce{HCO3^-}$ 或 $\ce{CO3^{2-}}$ 转化为难溶碳酸盐除杂。
      
      $$
      \ce{R^{2+} + 2HCO3^- = RCO3 \downarrow + H2O+ CO2 \uparrow}
      $$
   
   3. 一价金属常将待提纯离子转化为沉淀而抛弃溶液

**务必记牢常见的沉淀**

#### 2. 氧化剂、还原剂

- 常见的氧化性物质
  ：$\ce{KMnO4、K2FeO4、K2Cr2O7、KClO3、HNO3、H2SO4(浓)、NO2、
  MnO2、Cl2、NaClO、H2O2}$ 等。
- 常见的还原性物质：$\ce{Na2S、H2S、SO2、KI、K2SO3、H2O2}$ （与过渡金属氧化物同时出现）
- $\ce{Cl、S}$ 的性质务必记得复习一下
1. 转化物质
2. 除杂

## 三. 核心反应条件的控制

- 工业需要考虑成本，实验不用特别考虑成本（除非这玩意特别贵）

### 1. 温度

目的：

1. 制化学反应速率【反应原理】
2. 促进反应向 ×× 方向进行【反应原理】
3. 促进或抑制水解【反应原理】
4. 促进物质或抑制物质分解或挥发【元素化合物的性质】
5. 使催化剂的活性达到最高【提高单位时间内的产率】【反应原理】
6. 控制物质的溶解性
7. 气化【分离】或者液化【凝华需要粗导管放阻塞；收集】物质
8. 防止副反应发生【反应原理，看题干】
9. 降温或减压，减少能源成本，降低对设备要求，达到绿色化学目的

套路： 控制温度在 T1-T2 之间两段答

- 温度低于 T1：反应速率慢；溶液状态；是否副反应
- 温度高于 T2：物质挥发/水解/分解；反应过于剧烈；能耗过大

### 2. pH

反应原理上：反应速率、平衡移动、水解等  
物质性质上：离子沉淀、同价不同形、胶体稳定性、氧化或还原性等  
经济角度上：后续中和酸、碱的处理调节

相关内容：

1. 同价不同形（$\ce{Cr2O7^{2-}}$ 和 $\ce{VO3^-}$ 
2. 稳定性（$\ce{K2FeO4}$ 仅在碱性条件下稳定存在）

调节目的：

1. 杂质完全沉淀而产物不~~沉淀~~【查题干的 Ksp】
2. 确定溶质存在的形态（同价异形）

调节试剂：
**目标元素对应的氧化物**、**氢氧化物和碳酸盐** 

- 需要可以与 $\ce{H^+/OH^-}$  反应，使得 pH 变化
- 不~~引入新杂质~~

### 3. 试剂用量

结合一物多性和分离成本考量

1. 加入过量XXX的优点：
   
   1. 使XX完全沉淀
   2. XX成本低/无毒无害
   3. 廉价反应物浓度增加促进另一种转化率提高

2. 加入过量XXX的缺点：
   
   1. 引发副反应【看题干】
   2. 降低产品纯度
   3. 导致分离成本/能耗增加

### 4.过滤

1. 趁热过滤：一般要滤液，增大XX的溶解度
2. 冷却过滤：一般要滤渣，降低XX的溶解度促使其结晶析出

#### 5. 常见的物质分离方法

1. **杂质转化法**
2. **沉淀法**
3. **气化法：易挥发易升华**
4. **浸出法：萃取和反萃取**
5. **氧化还原法**
6. **加热灼烧法**

## 四. 产品的获得

### 1. 纯物质溶液获得晶体的方法

1. 蒸发结晶（产品不含结晶水）**【当蒸发皿中出现大量固体时停止加热】**
2. 冷却热饱和溶液：蒸发浓缩、冷却结晶（产品含有结晶水）
3. 醇析：溶液中加入醇溶液促使晶体析出（没有无机盐）
4. 盐析：溶液中加入某些盐或盐溶液，促使晶体析出【蛋白质、硬脂酸钠；同离子效应】

### 2. 从混合溶液获得晶体的方法【分离】

1. 得到<u>溶解度受温度影响大</u>的晶体  
   
   - 步骤：蒸发浓缩【富集】、冷却结晶、过滤、洗涤、干燥（减压/低温防止结晶水损失）
   - 注意：冷却过饱和溶液而不结晶时应：
     1. 加入晶核(种)
     2. 搅拌：用玻璃板轻轻摩擦烧杯内壁 
   - 蒸发浓缩停止加热时间：蒸发皿中刚出现固体时停止加热

2. 得到<u>溶解度受温度影响小</u>的晶体
   
   - 步骤：蒸发浓缩、趁热过滤【过液:
     杂质 ；固体 :⽬标产物
     】、洗涤、干燥。 

### 3. 对已获得晶体提纯的方法：重结晶

1. 选择适宜的溶剂，将粗产品制成热的饱和溶液
2. 趁热过滤，除去不溶性杂质（包括活性炭脱色）
3. 冷却结晶
4. 抽滤，滤出溶质
5. 洗涤，除去附着物和溶剂
6. 干燥

### 4. 含结晶水的物质脱去结晶水的方法

水解（弱碱金属阳离子）产生挥发性气体的：在气流中加热

#### 5. 洗涤晶体

1. 冰水洗涤：洗去晶体表面的杂质离子，减少晶体在洗涤中的溶解损耗，增大产率

2. 乙醇洗涤：增大杂质的溶解度，降低产物的溶解度，减少损失。  
   乙醇洗涤优点： 
   
   1. 酒精易溶于水、
   2. 且易挥发、
   3. 减少晶体溶解。

3. 浓盐酸洗涤：同离子效应

### 6. 干燥晶体

1. 无~~腐蚀~~、无~~挥发性~~、加热不~~分解~~及无~~结晶水的物质~~：烘干。

2. 热稳定性较差或含结晶水的物质：低温烘干/减压蒸发。  
   原理：减压蒸发降低了溶剂的蒸发温度，可以防止结晶水分解。

3. 还原性较强的晶体：真空干燥（极致减压）。  
   原理：促使溶剂快速挥发及创造真空环境，隔绝空气，防止晶体被空气中的$\ce{O2}$氧化。

## 五. 中肯地评价

### 1. 评价方案：往绿色化学去套

1. 能量高，污染小
2. 目标原子利用率高（增产减损）
3. 节约用量(试剂循环使用)
4. 操作简单
5. 安全

### 2. 工艺流程需要注意两大“解”

1. 分解（带结晶水或高温易分解的物质）。  
   操作：控制温度，降低反应剧烈程度、减压(蒸发/蒸馏)。
2. 水解（弱碱金属阳离子）。  
   操作：加入过量的酸/碱溶液。

### 3. 工艺流程需要注意两个“防”

1. 防挥发。  
   操作：控制温度。
2. 防氧化。  
   操作：煮沸（排除溶解的 $\ce{O2}$ ）、排尽装置中的空气、加入还原剂。

### 4. 证明物质已经分解完全或洗涤干净

**连续两次加热或洗涤后，称量物质的质量差不超过0.1 g，说明物质
已经分解完全或洗涤干净。**

## 5.1 任意角和弧度制

> 平面内一条射线绕其顶点从一个位置旋转到另一个位置所形成的图形叫做角（angle）

### 5.1.1 任意角

> 正角：按逆时针旋转所形成的角；  
>  负角：按顺时针旋转所形成的角；  
>  零角：没有做任何旋转．

$$
180 ^。= \pi rad \qquad 1 r a d = ( \frac { 180 } { \pi } ) ^ { \circ } \approx 57.3 ^ { \circ } = 57 ^ { \circ } 18 ^ { \prime }
$$

> **所有与角 $\alpha$ 终边相同的角，连同角 $\alpha$ 在内，可构成一个集合**

$$
S = \{ \beta | \beta = \alpha + k \cdot 360 ^ { \circ } , k \in Z \}
$$

> **即任一与 角 $\alpha$ 终边相同的角，都可以表示成角 $\alpha$ 与整数个周角的和**

### 5.1.2 弧度制

> $1$ 度的角等于周角的 $\dfrac{1}{360}$ 这种用度作为单位来度量角的单位制叫做角度制长度等于半径长的圆
> 弧所对的圆心角叫做 $1$ 弧度 (radian) 的角，弧度单位用符号 rad 表示，读作弧度

- 半径为 $1$ 的圆即为单位圆

定义了弧度之后，我们就有了圆弧面积的算法：
$$
\boxed{S=\dfrac{1}{2}\alpha r^2}
$$

**般地，正角的弧度数是一个正数，负角的弧度数是个负数，零角的弧度数是 $0$**

如图，设 $\alpha$ 是一个任意角，它的终边与单位圆交点于点 $P(x,y)$，那么： 
 
> 纵坐标 $y$（$i$ 的长度） 叫做 $\alpha$ 的正弦，即 
>$$
y=\sin \alpha
>$$
> 横坐标 $x$（$j$ 的长度）叫做 $\alpha$ 的余弦，即 
>$$
x=\cos \alpha
>$$
> 比值 $\dfrac{y}{x}$ 叫做 $\alpha$ 的正切，即 
>$$
\dfrac{y}{a}=\tan \alpha
>$$

在角 $\alpha$ 的终边上任取一点 $P(\lambda,\nu )$ ，由相似形原理得

$$
\sin \alpha = \frac { v } { \sqrt { u ^ { 2 } + v ^ { 2 } } } , \cos \alpha = \frac { A } { \sqrt { u ^ { 2 } + v ^ { 2 } } } , \quad \tan \alpha = \frac { v } { u }
$$

从三角函数的定义中可以得到一些代数关系，包括三角函数的有界性、不同范围的角的正弦值与余弦值的大小关系、
包括后面的同角三角函数的基本关系式．这些内容在讲完三角函数的图象时，会从图象角度去理解，现在也可以
从单纯的三角函数定义的角度去思考．

$$
| HB | = | y | = | \sin \alpha | ; | OH | = | x | = | \cos \alpha | ; | AD | = | \tan \alpha |

设角 $\alpha$ 的终边与单位圆相交于点 $P(x, y)$ ，
过 $P$ 作 $P M \perp x$ 轴， $P Q \perp y$ 轴， $M, Q$ 为垂足，连结 $A P ， B P$ 
$$
\because y=\sin \alpha ， x=\cos \alpha ，在  \triangle O P M 中， |P M|+|O M|>|OP|
$$  
$$
\therefore \sin \alpha+\cos \alpha>1
$$
$$
\because S_{\triangle P O A}=\frac{1}{2}|O A| \cdot|P M|=\frac{1}{2} y=\frac{1}{2} \sin \alpha
$$
$$
S_{\triangle P O B}=\frac{1}{2}|O B| \cdot|P Q|=\frac{1}{2} x=\frac{1}{2} \cos \alpha
$$  
$$
S_{扇形 O A B}=\frac{1}{4} \pi \times 1^{2}=\frac{1}{4} \pi；S_{\triangle O A P}+S_{\triangle O P B}<S_{扇形OAB}
$$  
$$
\therefore \frac{1}{2} \sin \alpha+\frac{1}{2} \cos \alpha<\frac{1}{4} \pi
$$
$$\sin \alpha+\cos \alpha<\frac{\pi}{2}
$$  
$$
\boxed{1<\sin \alpha+\cos \alpha<\frac{\pi}{2}}.
$$

(2) 解：  
由三角函数定义结合三角函数线，在 $(0 ， 2 \pi)$ 内，
使 $\sin \alpha>\cos \alpha$ 成立的 $\alpha$ 的取值范围是 $\left(\frac{\pi}{4}, \frac{5 \pi}{4}\right)$.

**单位圆在 $y=x$ 以上的部分有 $\sin x > \cos x$**

**终边相同的角的同一三角函数的值相等**
- 这话反过来是不~~对的~~，有些三角函数值相同可能有 $\pm 2k\pi$ 和 $\pm k\pi$ 两种情况
$$
\boxed{
\left. \begin{array}  { l  }   \sin ( \alpha + k \cdot 2 \pi ) = \sin \alpha \\ { \cos ( \alpha + k \cdot 2 \pi ) = \cos \alpha } \\ { \tan ( \alpha + k \cdot 2 \pi ) = \tan \alpha } \\ { \text{其中} k \in Z } \end{array} \right.} \tag{1}
$$

### 5.2.2 同角三角函数的基本关系

$$
\boxed{\sin ^ { 2 } \alpha + \cos ^ { 2 } \alpha = 1} \tag{2}
$$

当 $a \neq k \pi + \frac { \pi } { 2 } ( k \in Z )$ 时
$$
\boxed{\frac { \sin \alpha } { \cos \alpha } = \tan \alpha} \tag{3}
$$
两个公式的证明可以从单位圆出发
>这就是说，同一个角 $\alpha$ 的正弦，余弦的平方和等干 $1$ ，商等于角 $\alpha$ 的正切
